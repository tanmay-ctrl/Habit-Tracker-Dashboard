## **Features**
* **No Database Needed:** Saves data using Browser LocalStorage.
* **Visual Tracking:** Includes a calendar view and an analytics graph (Google Charts).
* **Streak Counter:** Automatically tracks how many days in a row you complete a task.

## **Tech Stack**
* HTML5, CSS3, JavaScript (ES6)
* Libraries: FullCalendar.js, Google Charts
